This is a public repository of a ecommerce backend developed using springboot. 
The ecoomerce was developed in microservice architecture.
The services are :
1. Auth/User Service
2. Product Service
3. Cart Service
4. Order Service
5. Inventory service
6. cloud-gateway
7. eureka server
